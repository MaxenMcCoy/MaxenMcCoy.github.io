About

Currently attending New Hampshire Technical Instute (NHTI) for game development. At the moment, I'm working on two major projects, my senior capstone project and a personal project.


The senior capstone project I'm working on is a light hearted couch-multiplayer collectathon/3rd person shooter called "The NeighBOOhood". In it you and your siblings are out trick-or-treating on Halloween night, competing to get the most candy before your mother comes to pick you up at the end of the night.


My personal project is a sci-fi first person shooter game under the working title "Appetent Synapse". This project is inspired greatly by the "push forward combat" of Doom(2016), and draws a lot of inspiration from highly mobile FPS games. While it is an FPS, it's weapon focus is on melee combat, with the primary weapon of the game being a sword. My two foci of this project are creating a highly agile player controller with satisfying game feel, and creating a versatile behavior tree framework for making a diverse set of enemies.
