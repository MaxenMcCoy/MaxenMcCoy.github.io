Solo Developer - Final project for AG101 (Intro to Game Design) at NHTI
Demonstrated skills: Unity, C#, level design, pixel art



A simple metroidvania platformer (heavily inspired by Team Cherry's Hollow Knight) made for the final project of my first game development course at NHTI.

This project was developed over the last couple of months of the course. I created all the code and assets for this project. My goals for the project was to recreate, on a smaller scale, many of the movement mechanics of the game Hollow Knight. For production, I really focused on getting all the mechanics in place early on so that I would be able to properly dive into level design as soon as possible. Having all the mechanics in place before this point meant that I could really play around with those mechanics, and have the levels really take advantage of them.
